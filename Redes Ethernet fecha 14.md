Modelo jerarquico

Las redes Ethernet suelen seguir un modelo jerárquico de tres capas:
Core: La capa central, que conecta diferentes edificios o campus.  (Si tiene backcup) 
Distribucion:La capa de distribución, que conecta diferentes segmentos de red dentro de un edificio.  (Si tiene backcup) 
Acceso: La capa de acceso, que conecta los dispositivos finales a la red. (No tiene backcup) 

Escalabilidad: Este modelo permite una gestión más eficiente de redes grandes y complejas.
